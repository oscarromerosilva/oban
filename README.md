# oBank

A digital banking MVP that simulates the experience of a modern online bank.

## Key features

- **Financial dashboard** with a consolidated view of portfolio and recent activity.
- **Accounts list** with key information (account type, balance, currency, status).
- **Recent transactions history** associated with the primary account.
- **Basic profile management** (customer name and email) from the `Settings` screen.
- **Modern React/TypeScript architecture**, using hooks for banking data and a contemporary UI ready to be extended with more digital banking capabilities.
